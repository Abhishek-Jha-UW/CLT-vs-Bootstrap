# Inference Arena: CLT vs Bootstrap

**Inference Arena** is an interactive statistical inference tool that compares **Classical (CLT-based)** and **Bootstrap (empirical)** approaches for estimating population parameters.

This app helps users understand *when theory works—and when data-driven methods are more reliable*.
---

##  What This Tool Does

Upload your dataset and:

- Estimate statistics:
  - Mean
  - Median
  - Proportion
  - Percentiles

- Compare two inference approaches:
  - **Central Limit Theorem (CLT)**
  - **Bootstrap Resampling**

- Generate:
  - Confidence Intervals (default: 95%)
  - Distribution plots
  - CI comparisons

- Diagnose your data:
  - Sample size
  - Skewness
  - Outliers
  - Variability

- Get AI-powered insights:
  - Which method is more reliable?
  - Why?
  - Practical recommendations
---

## Why This Matters

In real-world analytics:

- CLT assumptions often break (small samples, skewed data)
- Bootstrap offers a **data-driven alternative**

This tool helps you:
- Understand the trade-offs
- Make better statistical decisions
- Communicate results clearly
